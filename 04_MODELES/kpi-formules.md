# KPI et formules GreenOps

## KPI carbone

```text
gCO2e / transaction = CO2e total / nombre de transactions
kWh / 1000 transactions = énergie totale / (transactions / 1000)
```

## KPI ressources

```text
Idle CPU = CPU alloué - CPU consommé utile
Taux idle = ressources idle / ressources allouées
```

## KPI retries

```text
Taux retry = nombre de retries / nombre de requêtes initiales
Surcoût retry = N × r × er
```

## KPI logs

```text
Logs / transaction = volume logs total / nombre transactions
```

## KPI DevSecOps

```text
Couverture Sonar = dépôts avec Sonar / dépôts actifs
Couverture Checkmarx = dépôts avec Checkmarx / dépôts actifs
```
