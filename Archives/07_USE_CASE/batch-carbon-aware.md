# Use Case — Batch carbon-aware

## Problème

Les batchs peuvent créer des pics de charge et consommer fortement CPU/DB/I/O.

## Modèles mobilisés

- Scheduling ;
- capacité ;
- intensité carbone ;
- analyse de sensibilité.

## Actions

- identifier les batchs déplaçables ;
- mesurer énergie / batch ;
- ordonnancer selon contraintes métier, charge système et intensité carbone ;
- suivre kWh / batch et gCO2e / batch.
