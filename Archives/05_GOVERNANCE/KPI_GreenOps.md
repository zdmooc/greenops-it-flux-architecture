# KPI GreenOps et DevSecOps

## GreenOps

- gCO2e / transaction
- kWh / 1000 transactions
- kgCO2e / application / mois
- ressources idle
- volume logs / transaction
- taux retry / timeout
- énergie par batch

## Performance / SRE

- latence P95/P99
- disponibilité
- throughput
- saturation CPU/RAM
- queue depth
- MTTR / MTBF

## DevSecOps

- % dépôts avec pipeline
- % dépôts avec SonarQube
- % dépôts avec Checkmarx
- % quality gates actifs
- dette critique restante
- délai moyen de correction critique
