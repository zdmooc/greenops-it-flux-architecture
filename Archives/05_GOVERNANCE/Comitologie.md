# Comitologie

| Instance | Fréquence | Objectif |
|---|---|---|
| Point squads | Hebdo | Suivi actions applicatives |
| Point production / infogérant | Hebdo ou bihebdo | Capacité, incidents, données infra |
| Revue architecture | Bihebdo | Arbitrages techniques |
| Revue GreenOps | Mensuelle | KPI carbone, gains, trajectoire |
| COPIL | Mensuelle | Décisions, risques, priorités |

## Reporting mensuel

1. Synthèse exécutive : trajectoire, gains, risques, décisions.
2. GreenOps : CO2, ressources, logs, retries.
3. DevSecOps : SonarQube, Checkmarx, quality gates, dette.
4. Roadmap : actions, dépendances, arbitrages.
